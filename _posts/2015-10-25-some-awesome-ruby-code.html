  1. Store method in variable( Ruby functional programming properties) using method

    {% highlight ruby %} #If you want to store a method rather than the result of calling a method #or just the message you'd send to invoke it, you need to use the method method on the owning object. #So for example "hello".method(:+) #will return the + method of the object "hello", so that if you call it with the argument " world", #you'll get "hello world". helloplus = "hello".method(:+) helloplus.call " world" # => "hello world" {% endhighlight %}
