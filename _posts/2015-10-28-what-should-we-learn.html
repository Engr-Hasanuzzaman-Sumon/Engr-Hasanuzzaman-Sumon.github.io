এটা একটা ধারাবাহিক পোস্টঃ প্রথম পর্ব

আমরা এই পোস্টটি তাদের জন্য যারা নিজেদেরকে Software Engineer দেখতে চাও কিন্তু বুঝতে পারছনা কই শিখতে হবে, কোন Programming Language এর চাহিদা বেশি ......? আমরা ব্যাক্তিগত অভিজ্ঞতা থেকে বলছি আমরা অনেকে কি শিখব, কোনটার চাহিদা বেশি সেটা নিয়ে চিন্তা করে সময় পার করি (আমরা যেটা হয়ছে )। বর্তমানে কি শেখা দরকার বা কি কি আছে আমরা মতামত এখানে তুলে ধরার চেষ্টা করব। (আমরা ব্যাক্তিগত মতামত যা একদম ঠিক নাও হতে পারে )

প্রথমে আসি ওয়েব ডেভেলপারদের জন্য কি শেখা উচিৎঃ

১। HTML(HyperText Markup Language): কেউ যদি ওয়েব ডেভেলপারদের হতে চাই প্রথমে যেটা শেখা উচিৎ বলে আমি মনে করি সেটা HTML কাড়ন আমরা browser যা দেখি সেটা HTML আর CSS ছাড়া আর কিছুনা। সুতরাং আমাদের আসল উদ্দেশ্য HTML বানান (সেটা হতে পারে static html বা html generated by programming language like PHP, Java, Python, Ruby...)। *** browser আসলে html, CSS ছাড়া কিছু দেখাতে পারেনা আর JavaScript ছাড়া কছু run করাতে পারেনা বাকি সব Server Side programming language কাজ html generated করা আর কিছুনা *** সুতরাং আমদের অবশই ভালভাবে html জানতে হবে যেটা আমরা জানতে চাইনা (আমি মনে মনে করতাম যারা programming ভাল করেনা /পারেনা তারা html শিখবে যেটা ছিল চরম ভুল বা আমি আসলে ভালভাবে জানতামইনা আসলে html কাজ )

২। অবশ্যই CSS(Cascading Style Sheets যেটা style sheet language) যা দিয়ে markup language( যেমন HTML) এই ডকুমেন্টকে সুন্দর করে সাজাতে পারব।
