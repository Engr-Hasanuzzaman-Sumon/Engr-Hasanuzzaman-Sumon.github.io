  • Display git log information nicely
    git log --oneline --decorate --graph where --oneline show commit in one line, --decorate give some color and --graph gives graph show.
  • Display git branch
    Show all locals and remote branchs git branch -a Show all locals branchs only git branch Show all remote branchs only git branch -r
  • Remove/delete git branch from remote. You can also use the following to delete the remote branch. git push --delete origin serverfix Which does the same thing as git push origin :serverfix but it may be easier to remember.
  • Remove/delete git branch from local: You can also use the following to delete the local branch. git branch -d branch_name or for force delete git branch -D branch_name
